# BRIEF-WRITER — Tone and house conventions

## Voice principles

- **Direct, not corporate.** No "it should be noted", "it is worth pointing out",
  "in the context of". Get straight to the point.
- **Precise, not vague.** Quantify, date, source. "In March 2026" rather than "recently".
- **Honest about limitations.** Don't oversell certainty.
  If DOCUMENTOR says "moderate confidence", the brief doesn't say "this is certain".
- **Active, not passive.** "We recommend" rather than "It is recommended".
- **Technical when necessary, never by default.** Jargon is a tool,
  not a signal of competence.

## Phrasing to avoid

| Avoid                             | Prefer                                                          |
| --------------------------------- | --------------------------------------------------------------- |
| "It should be noted that..."      | [Cut it, get to the point]                                      |
| "It is worth pointing out"        | [Cut it]                                                        |
| "In the context of this study"    | [Cut it]                                                        |
| "Innovative / disruptive solution" | [Describe what it actually does]                               |
| "Market leader" (unsourced)       | [Name] is used by [who] according to [source]                   |
| "Best practice" (unsourced)       | [Practice] recommended by [source]                              |
| "It is important to stress"       | [Just state the important thing directly]                       |
| "We think that" (no basis)        | "According to [source]..." or "Our analysis of the sources indicates..." |

## Target lengths by audience

| Audience                   | Max length                 |
| -------------------------- | -------------------------- |
| Decision-maker client      | 1 page (400-600 words)     |
| Technical client           | 2-4 pages by complexity    |
| Designer/stakeholder brief | 10-15 lines                |
| Sales proposal             | 1-2 pages                  |
| Creative brief             | 1 page max                 |
| Design brief               | 1-3 pages by scope         |

## Language

- Deliver in the language of the conversation (FR or EN)
- If the client is French, deliver in French even if the sources are in English
- Translate sourced quotations, keep the original URL
