# Category: design / build

**Scope.** Create interfaces from scratch, translate designs into production code, generate design prompts, and set up per-project design context. These skills *produce* UI or the inputs to it. To improve an existing UI, use `design/refine/` instead.

**Naming.** Descriptive of the artifact or action — `frontend-design`, `mobile-responsive`, `figma-implement-design`, `design-prompt-generator`, `teach-impeccable`.

**Tone & audience.** Senior product designer / frontend engineer. Production-grade output, avoids generic AI look, accessible in light and dark.

**Required scaffolding.** `references/` for design systems, Figma-MCP workflows, or prompt templates as needed.

**Default frontmatter.**

```yaml
name: <folder-name>

user-invokable: true
```
