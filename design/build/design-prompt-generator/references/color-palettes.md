# HSL color palettes

Palettes ready to drop into a prompt. Each palette contains the complete set of shadcn/ui tokens.

Choose the palette based on the requested mood. You can mix: take the background from one palette and the primary from another, as long as the contrast stays sufficient.

---

## Useful additional tokens

Add these tokens when the hero needs specific text colors:
```
--hero-heading: [H] [S]% [L]%;
--hero-sub: [H] [S]% [L]%;
```

For text gradients, use an inline background-image rather than a token:
```
background-image: linear-gradient(223deg, #E8E8E9 0%, #3A7BBF 104.15%)
bg-clip-text text-transparent
```
